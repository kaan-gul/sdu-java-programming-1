# SDÜ Programlama 1 - Java Uygulama ve Ödev Arşivi

Bu depo, Süleyman Demirel Üniversitesi Programlama 1 dersi süresince işlenen **ders içi örnekleri** ve öğrencilere pratik yapmaları için verilen **çalışma sorularının/ödevlerin** çözümlerini içermektedir.

## 📖 İçerik Hakkında

Depo iki ana odak noktasına sahiptir:
1. **Ders İçi Örnekler:** Teorik konuların pekiştirilmesi için derste yazılan temel kodlar.
2. **Çalışma Soruları (Ödevler):** Konu sonlarında verilen uygulama materyallerinin detaylı çözümleri.

## 📂 Klasör Yapısı

Projeler, konularına ve içerik türüne göre organize edilmiştir:
* `src/KosulluIslemler/`: If-Else ve Switch-Case yapıları.
* `src/Donguler/`: (Yakında) For, While ve Do-While örnekleri.
